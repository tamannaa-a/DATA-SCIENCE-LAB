## Aim of the Lab

The aim of this Data Science Lab is to provide hands-on experience with fundamental and applied concepts of data science using **R and Python**.  
The lab focuses on data manipulation, exploratory data analysis, statistical analysis, machine learning techniques, and visualization to solve real-world data problems.

Through these experiments, students gain practical exposure to:
- Data handling and preprocessing
- Statistical reasoning
- Machine learning model building
- Visualization and interpretation of results
- End-to-end data science workflow
